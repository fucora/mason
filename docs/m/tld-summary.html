m

Standard Syntax:
     <%@ taglib prefix="m" uri="http://xml.metamug.net/jsp/jstl/mason" %>
XML Syntax:
     <anyxmlelement xmlns:m="http://xml.metamug.net/jsp/jstl/mason" />
 
REST API JSP Tag library.

Tag Library Information
Display NameNone
Version3.4
Short Namem
URIhttp://xml.metamug.net/jsp/jstl/mason
 

Tag Summary
resourceRest API resource tag for wrapping request tag
requestFor handling incoming requests with different HTTP methods
exceptionUsed for handling JSP exceptions on the error page. As defined by errorPage attribute of tag directive
headerAdditional response header to be sent along with the response payload.
paramDefine incoming request parameters. It helps in documenting and validating request parameters. For more information, visit https://metamug.com/docs/request-parameters.
scriptExecute a groovy script
executeExecute a Java RequestProcessable Class.
argArgument for the Execute tag. To pass configuration to the code being executed
xparamParameters to be sent in the request. Can be sent with GET as well.
xbodyRepresents the request body of an external api call.
xrequestXRequest allows the developer to make an HTTP request to external (3rd party) APIs. The tag can be placed anywhere inside the request tag.
downloadThis tag was previously used for downloading files. It is now deprecated. Instead perform file download with execute tag.
 

Function Summary
java.lang.StringjsonPath( java.lang.String,java.lang.Object)This tag is deprecated in favour of JSP EL with POJO. This was previously used to access JSON content
